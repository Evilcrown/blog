## 文件操作基础命令



- ls 文件列表

  - a (all) 显示全部文件 <font color=red>linux中文件命名前加. 代表隐藏文件</font>
  - l (long) 显示文件长信息
  - i (id) 显示文件id
  - h 友好方式显示文件信息，会计算文件大小为kb或者Mb
  - d (direct) 显示目录信息

  

- mkdir 目录创建

  - p 递归创建目录 创建多级目录

  

- cd 目录切换

  

- pwd （print working directory）打印当前目录

  

- rmdir （remove empty directories）删除空目

  

- cp （cope）复制文件

  - r  复制目录
  - p  保留文件属性

  

- mv （move）移动文件

  

- rm  （remove）删除文件或文件夹

  - r 递归删除 删除目录
  - f  强制删除



- touch 创建文件



- cat 查看文件 tac （反向查看文件）
  - n  显示行号



- more 查看文件
  - enter  按行向下翻页
  - space  按页向下翻页
  - q  退出
